Simple Java API for sending SMS,
This API support Twilio

# Maven Integration #

```
<!-- SMS API -->
        <dependency>
            <groupId>com.tchepannou.sms</groupId>
            <artifactId>tch-sms-api</artifactId>
            <version>1.0</version>
        </dependency>

<!-- Twilio Implementation -->
        <dependency>
            <groupId>com.tchepannou.sms</groupId>
            <artifactId>tch-sms-twilio</artifactId>
            <version>1.0</version>
        </dependency>

<!-- Dynamark Implementation -->
        <dependency>
            <groupId>com.tchepannou.sms</groupId>
            <artifactId>tch-sms-dynamark</artifactId>
            <version>1.0</version>
        </dependency>

<!-- Mock Implementation -->
        <dependency>
            <groupId>com.tchepannou.sms</groupId>
            <artifactId>tch-sms-mock</artifactId>
            <version>1.0</version>
        </dependency>
```
